## 컴포넌트

### 1. 주문/결제 취소 버튼
- 주문/결제 과정을 중단할 수 있는 버튼
- X 버튼 클릭 시, 결제 페이지 종료되고 이전에 보고있던 구매 페이지로 이동

### 2. N Pay 포인트/머니 표시 영역
- 상품 구매 시 사용 가능한 포인트와 머니가 표시되는 영역
- 아래의 기능으로 영역 내에서 구분됨
- 잔여 포인트/머니 표시
- 잔여 포인트/머니 사용
- 내 숨은 포인트 찾기

#### 2-1. 사용 가능 포인트 도움말 버튼
- 물음표 버튼 클릭 시, 포인트와 머니에 대한 설명이 적힌 모달 하단에서 노출

#### 2-2. 잔여 포인트 표시 영역
- 현재 보유하고 있는 잔여 포인트를 표시하는 영역

#### 2-3. 포인트 사용 영역
- 포인트 사용을 위해 액수를 입력할 수 있는 텍스트 필드
- 텍스트 필드 가장 우측 텍스트 '사용'은 플레이스홀더처럼 고정
- 텍스트 필드 내부 클릭 시, '원' 앞에 입력 커서 표시
- 숫자 입력 시, 녹색(#09AB49)으로 입력
- 보유 포인트 양보다 큰 액수 입력 시, '보유 금액 이상 사용은 불가능 합니다.' 팝업창 노출
- 팝업창에서 '확인' 버튼 클릭 시, 텍스트 필드에 입력한 숫자 자동 삭제

#### 2-4. 전액 사용 버튼
- 보유하고 있는 포인트를 구매에 전액 사용할 수 있는 버튼
- 버튼 클릭 시, [2-2]에 표시된 금액이 [2-3] 텍스트 필드에 자동으로 입력됨

#### 2-4a. 전액 사용 버튼 적용 시 UI
- 전액 버튼 클릭 시, 전액 사용 버튼 디폴트 상태로 변경

#### 2-5. 항상 전액사용 버튼
- 구매 시 자동으로 포인트가 전액 사용되도록 설정하는 버튼
- 텍스트 필드 작성 상태에 따른 버튼 작동 방식
- a. [2-3] 텍스트 필드에 숫자가 입력되지 않은 상태(0원)
- '항상 전액사용 버튼' 클릭 시 자동으로 보유 포인트 사용되고, [2-4a]와 같이 '전액사용 버튼' 디폴트로 변경
- b. [2-3] 텍스트 필드에 숫자가 입력된 상태
- '항상 전액사용 버튼' 클릭 시 입력된 액수가 보유 포인트 액수로 변경되고, [2-4a]와 같이 '전액사용 버튼' 디폴트로 변경
- '항상 전액사용 버튼'을 클릭하여 활성화 된 상태에서 재클릭 시, 버튼 클릭 상태만 해제되고 [2-3] 텍스트 필드에는 변화없음

#### 2-6. 내 숨은 포인트 찾기 버튼
- 추가적으로 수령할 수 있는 포인트를 확인하도록 만드는 UI
- > 버튼 클릭 시, 제휴포인트 전환 페이지로 이동 (리디자인 영역 X)

### 3. 후불 결제 영역
- 후불 결제 서비스를 이용할 수 있도록 안내하는 UI
- 후불 결제 좌측의 체크 아이콘 클릭 시,
- 미신청 유저:후불 결제 신청 모달창 하단에서 노출(리디자인 영역 X)
- 신청 유저: 후불 결제 적용 UI로 변경
- [3] 후불 결제 왼쪽 체크 아이콘 클릭 시, 변경되는 UI 참고

### 4. 결제 적립 포인트 안내 영역
- 해당 구매건을 통해 적립되는 포인트를 안내하는 영역
- 구매적립 영역
- 기본적립: 네이버페이 포인트를 사용하지 않고 구매 시 적립되는 포인트 액수 표시
- 네이버페이 머니 결제적립: 네이버페이 포인트를 사용해 구매 시 적립되는 포인트 액수 표시
- 리뷰적립 영역
- 해당 구매건의 리뷰 작성 시 적립받을 수 있는 예측 포인트 액수 표시
- 리뷰 적립 포인트 비율 설정 필요

#### 4-1. 네이버 멤버십 혜택 표시 영역
- 네이버페이 멤버십 가입 시, 제공받을 수 있는 혜택을 확인할 수 있는 버튼
- > 버튼 클릭 시, 네이버페이 멤버십 혜택 페이지로 이동

### 5. 개인정보 제공동의 영역
- 구매 진행 시, 네이버페이의 이용 약관에 동의한다는 것을 안내하는 영역

#### 5-1. 개인정보 제공동의 약관 표시 버튼
- 상세보기 버튼 클릭 시, 개인정보 제공 동의 약관이 기재된 팝업창 화면 중앙에 노출

#### 5-2. 구매안전 서비스 안내 버튼
- ? 버튼 클릭 시, 하단에 구매안전 서비스 안내 모달창 노출

### 6. 충전 후 결제하기 버튼
- 결제하기 버튼 클릭 시, 선택한 결제 수단으로 결제하는 과정으로 진입
