## 컴포넌트

### 1. 뒤로가기 버튼
- 클릭시 네이버 플러스 스토어 메인 화면 진입

### 2. 검색 버튼
- N배송 카테고리 필터가 적용된 상품을 검색할 수 있는 버튼

### 3. 장바구니 버튼
- 클릭시 장바구니로 이동

### 4. 주소 변경 버튼
- 주소 변경 버튼
- 배송지 근처 NFA(Naver Fullfillment Alliance) 물류센터 입고 여부/거리 등에 따라 상품 및 배송 정보가 변경될 수 있음.

### 5. 내일배송 상품 개인화 추천
- '내일배송' 상품 중, 사용자의 관심사와 연관도가 높은 상품들 위주로 배치되는 영역

#### 5-1. 소제목
- 개인화 맥락을 반영을 강조한 소제목 표시

#### 5-2. 가로 스크롤
- 좌우 스크롤로 더 많은 상품 탐색 가능

#### 5-3. 내일배송 강조 영역
- 로고를 활용해 '내일배송' 상품임을 강조하는 영역

#### 5-4. 더보기 버튼
- 내일배송 카테고리 필터 적용된 상품 페이지로 전환

### 6. 검색 버튼
- N배송 카테고리 필터가 적용된 상품을 검색할 수 있는 버튼
- 3번과 동일 기능

### 7. 카테고리 구조 개선
- 사용자 맞춤형 카테고리 상위 10개만 우선적으로 표시
- 나머지는 우선 숨김 처리

#### 7-1. 전체 카테고리 더보기
- 클릭 시 총 25개의 세부 카테고리까지 모두 노출

#### 7-1a. 전체 카테고리 닫기
- ^ 클릭 시 상위 카테고리만 남기고 hide
- 7-1의 화면으로 전환

### 8. 당일배송 안내 배너
- 당일배송 마감 시각 강조

#### 8-1. 당일배송 상품 목록 이동 버튼
- > 버튼 클릭 시 당일배송 카테고리 필터 적용된 상품 목록으로 전환

### 9. 타임딜 섹션
- 시간 제한 특가 상품을 제시하는 프로모션 영역

#### 9-1. 카운트다운 컴포넌트
- 카운트다운 타이머를 시각화하여 사용자의 구매 긴급성을 유도하는 디스플레이
- 시간은 HH:MM:SS 형태로 구분

#### 9-2. 프로모션 태그
- 브랜드데이, +세일 등 프로모션 종류를 강조하는 태그

### 10. 메인 배너
- N배송 핵심 메시지 전달 및 브랜드 가치 강조 배너
