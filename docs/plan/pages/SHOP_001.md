## 컴포넌트

### 1. GNB
- 상단 네비게이션 영역

#### 1-2. 뒤로가기 버튼
- 탭 시 이전 화면으로 이동

#### 1-3. 주문배송 버튼
- 탭 시 주문배송 화면으로 이동

#### 1-4. 마이쇼핑 버튼
- 탭 시 마이쇼핑 화면으로 이동

### 2. 배송 탭
- 일반배송 탭: 선택 시 일반배송 상품 목록 표시
- 컬리N마트/지금배달 탭: 선택 시 해당 상품 목록 표시

### 3. 배송지 영역
- 현재 설정된 배송지 주소 표시

#### 3-1. 변경 버튼
- 탭 시 배송지 변경 화면으로 이동

### 4. 전체선택/선택삭제 영역
- 전체선택 체크박스: 전체선택 체크박스 클릭 시 장바구니 내의 전체 상품 선택/해제
- 선택삭제 버튼: 선택된 상품 일괄 삭제
- 선택된 상품이 없을 경우 버튼을 클릭해도 인터랙션 X

### 5. 개별 상품 표시 영역
- 장바구니에 담은 개별 상품 표시 영역
- 스토어명, 배송 도착 예정일 정보, 상품명, 내일 배송을 위한 잔여 시간 타이머, 가격, 상품 개수 조정, 옵션 변경으로 구성

#### 5-1. N쿠폰받기 버튼
- 해당 스토어에서 구매하는 상품에 대한 쿠폰을 다운받을 수 있는 기능
- 쿠폰 받기 클릭 시 해당 스토어 쿠폰 다운로드 창으로 팝업창 노출
- 팝업창 (리디자인 영역 X)

#### 5-2. 삭제 버튼
- 해당 상품을 장바구니에서 삭제할 수 있는 버튼
- X 버튼 클릭 시, 해당 상품 즉시 장바구니에서 삭제

#### 5-3. 상품 개수 조정 버튼
- 해당 상품의 구매 개수를 선택할 수 있는 기능
- + / - 버튼 클릭을 통해 개수 조정 가능
- 1개 이하로는 - 버튼 클릭되지 않음
- 1개에서 - 버튼 클릭 시, 1개 이하는 구입할 수 없다는 알림창 노출

#### 5-4. 옵션 변경 버튼
- 상품의 옵션을 변경할 수 있는 버튼
- 옵션 변경 버튼 클릭 시, 하단의 옵션 변경 모달 올라옴
- 모달 내부의 옵션 클릭 후 적용 시, 해당 옵션으로 상품 상세 내용 변경 (리디자인 영역 X)

#### 5-5. 카운트다운 타이머
- 당일 주문 마감까지 남은 시간을 표시하는 타이머
- 실시간으로 시간이 카운트다운됨
- 마감 시간이 임박할 시, 빨간색으로 시간이 강조됨

### 6. 주문 예상 금액 요약 영역
- 좌측 상단 체크박스를 통해 선택한 상품의 주문 예상 금액을 확인할 수 있는 영역
- 총 선택상품금액: 선택한 상품의 총 합계를 합산할 수 있는 기능
- 즉시할인예상금액: 선택한 상품 중 즉시할인이 가능한 상품이 있다면 할인된 금액을 표시하는 기능
- 쿠폰할인예상금액: 선택한 상품 중 쿠폰을 통해 할인을 적용한 상품이 있다면 할인된 금액을 표시하는 기능
- 총 배송비: 선택한 상품 중 배송비를 별도로 부과하는 상품이 있다면 배송비를 표시하는 기능
