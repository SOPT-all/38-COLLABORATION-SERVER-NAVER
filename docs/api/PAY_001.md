# PAY_001 결제 API 명세

---

## API 1. 포인트 조회

주문 ID로 사용 가능한 포인트 및 적립 예정 포인트 정보를 조회하여 반환한다.

**GET** `/api/orders/{orderId}/points`

### Path Parameter

| Name | Type | Required | Description | Example |
|------|------|----------|-------------|---------|
| orderId | Long | Y | 주문 ID | `1` |

### Query Parameter


(없음)

### Request Header

(없음)

### Request Body

(없음)

### Request Example

```
GET /api/orders/1/points
```

### Response Body

| Name | Type | Required | Description | Example         |
|------|------|----------|-------------|-----------------|
| success | Boolean | Y | 요청 성공 여부 | `true`          |
| status | Integer | Y | HTTP 상태 코드 | `200`           |
| message | String | Y | 응답 메시지 | `"요청이 성공했습니다."` |
| data | Object | Y | 포인트 조회 데이터 | -               |
| data.orderId | Long | Y | 주문 ID | `1`             |
| data.availableAmount | Integer | Y | 사용 가능한 총 금액 | `87`            |
| data.availablePoint | Integer | Y | 사용 가능한 포인트 | `80`            |
| data.availableMoney | Integer | Y | 사용 가능한 머니 | `0`             |
| data.hiddenPoint | Integer | Y | 숨은 포인트 금액 | `7`             |
| data.allUseAmount | Integer | Y | 전액 사용 시 적용 금액 | `87`            |
| data.deferredPaymentAvailableAmount | Integer | Y | 후불결제 이용 가능 금액 | `300000`        |
| data.expectedPoint | Object | Y | 적립 예정 포인트 정보 | -               |
| data.expectedPoint.totalPoint | Integer | Y | 총 적립 예정 포인트 | `1472`          |
| data.expectedPoint.purchasePoint | Object | Y | 구매 적립 포인트 정보 | -               |
| data.expectedPoint.purchasePoint.total | Integer | Y | 총 구매 적립 포인트 | `722`           |
| data.expectedPoint.purchasePoint.basicPoint | Integer | Y | 기본 적립 포인트 | `289`           |
| data.expectedPoint.purchasePoint.npayMoneyPoint | Integer | Y | 네이버페이 머니 결제 적립 포인트 | `433`           |
| data.expectedPoint.reviewPoint | Integer | Y | 리뷰 적립 포인트 | `750`           |
| data.expectedPoint.membershipPoint | Integer | Y | 멤버십 추가 적립 포인트 | `1156`          |
